# advent-of-code-2022

## Setup

1. Install Nix
2. Run `nix develop`
3. Run `zigmod fetch`

## Run a solution

Choose one of the number of solutions from the `day` directory, and run the
following where `${DAY_NUMBER}` is substituted with the number you chose.

```bash
zig build run-day-${DAY_NUMBER}
```
